## Resumen

Estas APIs utilizan el flujo OAuth 2.0 para la autenticación.

Para comenzar, sigue los [pasos de autorización](docs/oauth/Authorization.md).

### Campos estándar de respuesta

A continuación se listan los campos estándar que recibirás en cada respuesta.

#### TraceId

Un traceId es un identificador único por solicitud y se devuelve en cada respuesta. Es útil para localizar la solicitud exacta y para depuración.
